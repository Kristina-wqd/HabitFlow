# HabitFlow — Трекер привычек

## Сайт доступен по адресу:
https://kristina-wqd.github.io/HabitFlow/

## О проекте
- **Тема:** Приложение для трекинга привычек
- **CSS-фреймворк:** Bulma
- **Автор:** Ильясова Кристина


## Структура
1. **Навигация** — логотип и меню
2. **Герой-секция** — заголовок, описание, кнопки скачивания
3. **Особенности** — 3 карточки с ключевыми функциями
4. **Как работает** — 3 простых шага
5. **Тарифы** — бесплатный и премиум планы
6. **Футер** — контакты и социальные сети

## Технологии
- HTML5
- CSS3
- Bulma CSS Framework
- Font Awesome Icons
- JavaScript (для мобильного меню)

## Адаптивность
Лендинг полностью адаптирован для:
- Мобильных телефонов
- Планшетов
- Десктопов
